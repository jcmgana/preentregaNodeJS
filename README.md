Preentrega: Gestión de Productos FakeStore API

Este proyecto es una herramienta de línea de comandos (CLI) desarrollada en Node.js para gestionar productos consumiendo una API externa.
Comandos disponibles:
  - Obtener todos los productos:  npm run start GET products
  - Obtener un producto específico:  npm run start GET products/1
  - Agregar un nuevo producto:  npm run start POST products "Titulo" 100 "Categoria"
  - Eliminar un producto:  npm run start DELETE products/1
